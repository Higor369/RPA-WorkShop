# RPA-WorkShop
Roteiro de apresentação workshop RPA FNC

Requisitos
- Python 3
- pycharm ou outro editor da sua preferencia 
- google chrome

Dependencias
- rpa
- pyautogui
- pandas
- opencv-python
- openpyxl

Instalação de dependências
- abra o seu terminal 
- digite 'pip install' segida do nome da dependência
- ex: pip install pyautogui
